# 一张图扩展成一套视觉身份：简易复制版

上传一张能够代表你、品牌、角色或产品的图片，然后把下面整段复制给 AI。

```text
请把我上传的图片当作“视觉母体”，先判断它属于 Logo、人物头像、个人形象、IP 角色、插画、产品还是标志性物件，再提取其中可以持续复用的视觉基因。

请区分两类信息：
1. 必须保留：身份、核心轮廓、五官或几何比例、主色、标志性服装/道具、材质、品牌文字。
2. 可以变化：动作、表情、镜头、版式、背景、应用载体和传播场景。

基于同一套视觉规则，为我生成五条可以直接用于生图的完整提示词：
1. 品牌或个人主视觉；
2. 视觉识别系统展示板；
3. 角色动作、辅助图形或产品变化系统；
4. 包装、周边、数字产品或其他应用场景；
5. 社交媒体封面、海报或真实传播场景。

每条提示词都必须独立完整，明确写出需要保留的参考图特征、构图、色彩、材质、光线、文字、画幅和负面约束。不要写“同上”，不要留下括号占位符、斜杠菜单或未决定的选项。

先输出一张简短的“视觉 DNA 卡”，再输出五条提示词。最后检查五条提示词在身份、轮廓、色彩、文字和气质上是否一致。如果我没有提供品牌名或准确文字，不要自行编造。
```

## 更完整的 Skill

支持 Agent Skills 的工具可以安装整个 `visual-identity-expander/` 目录。完整版本会根据 Logo、人物、IP 和产品自动切换延展逻辑，并提供一致性检查与修正规则。

触发示例：

```text
用 visual-identity-expander，把我上传的头像扩展成一套个人品牌视觉提示词。
```

```text
用 visual-identity-expander，把这个 Logo 扩展成主视觉、辅助图形、包装和社交媒体视觉。
```
