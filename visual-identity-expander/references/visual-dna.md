# Visual DNA Rules

## 先分清两类特征

### 不可变特征

用于维持“还是同一个对象”：

- 人物：脸部身份、年龄区间、发型、眼镜、肤色、标志服装与配饰。
- Logo：几何比例、负空间、字标关系、主色和核心轮廓。
- IP：头身比、轮廓、五官比例、主配色、服装与标志道具。
- 产品：外形、结构、材质、接口、标签层级与品牌标记。

### 可变特征

用于产生足够丰富的应用：

- 动作、表情、视角、镜头距离。
- 背景、光线、版式、应用载体。
- 辅助色、纹样密度、场景道具。
- 海报、包装、社交封面、空间和周边的具体呈现。

## 输入路由

### Logo

把视觉基因重点放在形状语言、负空间、色彩、字体关系和重复方式。不要强行把 Logo 变成人物。需要 IP 时，先从 Logo 轮廓中提炼角色结构，并说明这是衍生角色。

### 人物头像或个人形象

脸部身份优先级最高。风格化只能改变媒介与画法，不能把人物变成另一张脸。服装和标志性物件可作为个人品牌识别点。

### IP 或插画

优先锁定轮廓和头身比。复杂背景可以放弃，但角色比例、五官和道具不能漂移。

### 产品或物件

先保证产品读得出来，再做品牌化。不要为了构图随意增加不存在的按钮、接口、包装文字或功能。

## 多参考图

- 指定一张主参考图锁身份。
- 其他图片只提供风格、色彩、材质或场景。
- 不把多个不同人物或角色平均融合成新对象，除非用户明确要求。
