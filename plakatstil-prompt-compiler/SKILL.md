# Plakatstil Prompt Compiler

把文字主题、上传商品图、包装图或人物服务主题，编译成**一条**可直接交给图像模型的 Plakatstil / Sachplakat 商品广告海报提示词。

核心原则：**风格锁死，方案放开；默认输出已决策的成品 prompt；用户要模板时，输出中文可改字段在前、英文固定风格在后的复制即用结构。**

## Read

按任务读取：

- 生成提示词：读 `references/compile-contract.md` 和 `references/module-library.md`。
- 判断像不像：再读 `references/evaluation-rubric.md`。
- 用户要示例或主题灵感：读 `references/examples.md`。

## Input

优先从用户输入推断，不要求填表：

1. **输入模式**：文字主题、商品照片、包装照片、人物/服务主题、风格评价。
2. **广告对象**：必须落到一个具体可销售对象。抽象概念要转译成物品。
3. **文字**：可包含中文短词、英文短词、品牌词、商品词、包装/标签字，以及可选的一行小副标题。用户指定中文时必须保留原中文，不要默认改成英文；不要强制只有一行字、两行字或固定位置。
4. **画幅**：优先遵循用户指定；未说明时根据对象、用途和构图自行判断。商品海报通常可用竖版，横向物体或包装可用横版，上传图可沿用原图比例。不要在 skill 中固定某个主题的比例。

## Workflow

1. **识别输入**：判断是文字生成、图片重绘、包装重绘、人物服务海报，还是风格评价。
2. **确定广告对象**：抽象主题先转成物品；商品照片保留轮廓、材质、标签层级和最可识别细节。
3. **选择模块**：从 `module-library.md` 选择主体、字体、版式、背景、色彩和纸墨质感。多张图时至少变化三类模块，优先变化物体隐喻、色彩、背景结构、文字布局和副标题是否出现，避免连续输出同一套版式。
4. **编译提示词**：按 `compile-contract.md` 输出。默认输出已决策的英文美术指导，不留下 `{}`、斜杠菜单、either/or 布局、未决选项。用户明确要求“复制即用模板 / reusable template / 文章提示词结构”时，输出中文输入字段在前、英文风格锚定在后的模板。
5. **输出**：已选模块 + 一条完整提示词 + 一句选型理由。用户只要 prompt 时，可以省略解释，只保留模块摘要和完整提示词。

## Routing Rules

- **简单小物件**：火柴、钥匙、笔、开关、耳机等，优先稀疏负空间和大手绘品牌字。
- **玻璃/金属/镜头/灯泡**：优先物体压住大品牌字、横向色带、手绘高光。
- **包装/标签**：肥皂、咖啡袋、鞋油、唱片、盒装商品，允许包装文字成为画面核心。
- **现代电子设备**：耳机、键盘、手表、手机、笔记本、移动电源、头显等可以用真实品牌、虚构品牌或无品牌处理；只有用户明确指定品牌时才锁品牌。默认不要强行套某个固定品牌，也不要做现代发布会海报。
- **人物/服务**：只能称为 Plakatstil-inspired service poster；严格 Sachplakat 仍以商品为主角。

复制即用模板：

````markdown
```text
文章主题或摘要：
{输入文章主题、标题、摘要，或直接粘贴文章内容}

用途和比例：
{例如：文章封面，横版 5:2}

{可选；没有就写：自动判断}

指定副标题文字：
{可选；没有就写：自动判断或不加}
````

模板里不要加入“你只需要改这里”“以下内容不用改”之类说明行。字段本身已经足够清楚。

## Quality Gates

- 必须是早期德国 Plakatstil / Sachplakat 商品广告逻辑，不是泛复古海报。
- 必须有一个明确商品或可销售对象。
- 必须是手绘风格：主体可以平面，也可以有立体感，但必须像手绘出来的广告物。
- 字体必须是手绘商业广告字：可以棱角分明，也可以柔和；可以粗重、块状、不规则；可以被物体遮挡、分开布局、放在物体上或包装上。
- 用户要求中文时，可以输出中文手绘标题字；中文建议 2-4 个汉字，必须使用用户指定原文，避免假中文、乱码字和随机标签。
- 可根据主题或用途增加一行小副标题。副标题只能一行，中文建议 6-12 个汉字，英文建议 2-6 个词；它必须明显小于主标题，像手绘广告小标签或窄条文案。
- 自由度优先：除非用户明确指定，不要固定某一种构图、配色、背景、标题位置或副标题位置。允许模型在清楚边界内自由决定最强的画面方案。
- 不限制颜色。颜色由对象、主题和构图决定，可以深色、亮色、对比色、复古色、品牌色或参考图取色，但必须像手绘/石版印刷颜料，不是现代干净渐变。
- 背景可以是纯色、纸色、横向彩条、色块、边框、包装色、图案或其他装饰，但必须服务于主体。
- 成组输出时，不能只换物体。每张图的主色、背景结构、标题位置、物体角度或装饰元素至少有三项明显不同。
- 纸张纹理、油墨吸收、套印误差和手绘边缘要像印刷材料，不是后期脏滤镜。
- 不编造已经生成的图片结果；本 skill 只产出提示词或评价。
