# Evaluation Rubric

Use this when judging whether an image matches Plakatstil / Sachplakat references.

## Score

Score each item 0-2.

### 1. Commercial Object Clarity

- 0: abstract icon, symbol, or concept image.
- 1: object exists but feels like illustration or modern poster prop.
- 2: one sellable object dominates the ad.

### 2. Plakatstil Typography

- 0: modern font, clean vector type, comic/rock title, or generic sans-serif.
- 1: hand lettering exists but does not feel like an ad wordmark.
- 2: custom hand-painted commercial lettering integrated with the product.

### 3. Composition Logic

- 0: generic centered poster or unrelated decorative layout.
- 1: strong layout but overuses one fixed formula.
- 2: object, text, background, and negative space form a clear product ad.

### 4. Palette Discipline

- 0: modern gradients, neon, or too many colors.
- 1: limited colors but forced into one repetitive palette.
- 2: restrained lithographic palette selected for the object.

### 5. Paper / Print Materiality

- 0: clean digital vector or photo render.
- 1: has noise/grunge but texture feels pasted on.
- 2: paper fiber, ink absorption, rough painted edges, and print imperfections feel integrated.

### 6. Content Freedom

- 0: style collapses into unrelated genre.
- 1: style is present but output is too rigid across subjects.
- 2: style is fixed while text, layout, palette, and background adapt to the product.

## Labels

- 10-12: close to Plakatstil / Sachplakat.
- 7-9: partially close; repair targeted modules.
- 4-6: generic vintage poster; not enough Plakatstil.
- 0-3: wrong visual system.

## Common Problems

- Too much icon language: looks like an app symbol, not an ad poster.
- Too much modern graphic design: Swiss grid, Bauhaus shapes, vector logo, clean sans-serif.
- Too much comic/pulp: aggressive title, dramatic light, heroic object angle.
- Too much fixed template: every image uses dark background, cream top word, red small word, and diagonal object.
- Texture added too late: paper grain appears as a filter instead of ink on paper.
- Copy too rigid: only one brand word plus one product word, even when packaging or label text should participate.

## Repair Pattern

1. Name the closest behavior: Priester sparse object, Osram object-over-brand, package/label hero, or cropped product macro.
2. Choose one layout module.
3. Choose one typography behavior.
4. Specify an object-derived or package-derived palette.
5. Add integrated print texture.
6. Remove modern/comic/vector negatives.
